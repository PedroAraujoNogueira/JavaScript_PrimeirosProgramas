    var ano = 2016;
    var puloLinha = "<br>";

    document.write("Flávio tem " + (ano - 1977) + " anos");
    document.write(puloLinha);
    document.write("Joaquim tem " + (ano - 1996) + " anos");
    document.write(puloLinha);

    ano = 2017;
    document.write("Barney tem " + (ano - 1976) + " anos");

</script>
Esse codigo permite alterar a quantidade de pulos de linha em um unico lugar e alterar em todos os lugares em que haja pulo de linha.

Como criamos uma função no JS? Convencionaremos que todas as funções que criarmos estarão nas primeiras instruções da tag <script>. Para declararmos uma função, utilizaremos o termo em inglês function, seguido de um nome.

Um ponto importante - a função é sempre descrita por um verbo, seja ele "pula", "grita", "comemora", ou qualquer outro -, isto porque ela sempre indica uma ação a ser executada.

<meta charset="UTF-8">

<script>

    function pulaLinha() {

        document.write("<br>");
        document.write("<br>");

}

var ano = 2016;

document.write("Flávio tem " + (ano - 1977) + " anos");
pulaLinha();

document.write("Joaquim tem " + (ano - 1996) + " anos");

pulaLinha();

ano = 2017;
document.write("Barney tem " + (ano - 1976) + " anos");

</script>
Funçao pulaLinha, usada para deixar o codigo anterior mais legivel e com maior possibilidade de funcionalidades.

Coloca-se trechos como comentários, por meio de barras duplas (//). O programa ignorará a referida linha.

Uma função pode ter como instrução a chamada de outra função.

<script>
    function pulaLinha() {

    	document.write("<br>");
    	document.write("<br>");
    }

    function mostra(frase ){

    	document.write(frase);
	pulaLinha();

    }

    mostra("Olá pessoal!");

    var ano = 2016;

    mostra("Flávio tem " + (ano - 1977) + " anos");

    mostra("Joaquim tem " + (ano - 1996) + " anos");

    ano = 2017;

    mostra("Barney tem " + (ano - 1976) + " anos");

</script>
